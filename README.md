# Half Birthday Calculator Website

A modern, React-based static website that calculates traditional and accurate half birthdays.

## Features
- **Traditional Calculation:** Adds exactly six months to the birthday.
- **Accurate Calculation:** Finds the halfway point in time between the current birthday and the next one.
- **Modern UI:** Responsive design with smooth "sweep" animations.
- **Tech Stack:** React (TypeScript), Vite, Vanilla CSS.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Testing
```bash
npm test
```

### Build
```bash
npm run build
```

## Deployment
Automated via GitHub Actions to GitHub Pages on every push to the `main` branch.

## Library
This project uses the `@seancrosby/half-birthday-calc` library.
